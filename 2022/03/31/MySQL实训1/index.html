MySQL实训1-MySQL的安装和配置

实训一 MySQL的安装和配置

一、实训目的与要求

了解MySQL数据库操作环境和其主要管理工具。

二、实训设备和软件环境

计算机、MySQL软件

三、实训课时

2课时

四、实训内容和结果记录

1. 从MySQL官网（http://dev.mysql.com/downloads/）下载MySQL数据库，解压缩后进行安装，观察安装成功后的效果。（也可以选择免安装版）

（1）操作步骤参照PPT—模块二相关内容。

（2）查看系统数据库。

2. 配置环境变量

（1）找到MySQL安装路径下bin文件夹打开，复制地址栏里的路径。

（2）“计算机”属性→“高级系统设置”→ “环境变量”→“新建”，输入变量名MySQL，变量值粘贴上一步骤复制的路径即可。

3．查看MySQL服务

（1）若服务未开启，在dos平台输入net start mysql启动

（2）net stop mysql停止服务

（3）打开cmd，mysql –u root -p进入MySQL控制台

4．常用命令练习

（1）显示数据库show databases;

（2）使用数据库use mysql;

（3）显示当前连接的数据库select database();

（4）显示当前服务器的版本select version();

（5）显示当前日期时间select now();

（6）显示当前用户select user();

（7）数据库详细信息status;

（8）显示系统可以设定的字符集show character set;

（9）显示某个字符集的校对原则（以utf8为例）show collation like 'utf8%';

（10）查看数据库的字符集show variables like 'character%';

（11）查看数据库中的表show tables;

（12）查看表结构（user是表名称）desc user;

五、实训思考题

1.第一次安装MySQL时，系统会自动建立4个数据库，分别是（ ）、sys、（ ）和（ ）这4个数据库是系统数据库。

2.MySQL的日志文件以及数据库文件放置在（ ）目录下（填默认位置）。

3.用于数据库用户信息和权限信息的数据库是（ ）。
