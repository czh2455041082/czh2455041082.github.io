实训三 数据表的创建和维护

数据表:

一、实训目的与要求

通过实训，要求学生掌握在使用MySQL语句和图形化界面（WorkBench或Navicat）创建和管理数据库及表的方法。(1)创建和管理数据库。(2)创建表、确定表的主码和约束条件。

二、实训设备和软件环境

计算机、MySQL软件、WorkBench或Navicat软件

三、实训课时

2课时

四、实训内容和结果记录

在前面的实训二中我们已经分别创建了Teacher、Class、Student、Course和Score数据表，如果还未创建数据表则先创建数据表，如果数据表中先前已经插入了数据则清空各数据表中的数据。接下来我们进行本次实训的任务：

1．修改表结构

⑴ 向学生表Student添加JoinSchoolTime（入学时间）列，此列数据类型为date。使用SQL语句完成此任务，语句如下：

alter table student add joinschooltime date after sex;

添加完成以后，请在确认是否已经添加了此列。

⑵ 删除学生表Student刚添加的JoinSchoolTime（入学时间）列，使用SQL语句完成此任务，语句如下：

alter table student drop joinschooltime;

删除完成以后，请在SSMS中确认是否已经删除了此列。

再用WorkBench或Navicat重复上面操作。

2．在WorkBench或Navicat可视化界面中在数据表中增、删、改数据

a. 向Teacher中插入如下数据：

TeacherNo Name
T01 张三
T02 李四

insert into teacher(teacherno,tname) values('T01','张三'),('T02','李四');

b. 修改第二条数据的Name字段值为”王五”。

select*from teacher; insert into teacher set teacherno='T03',tname='王五';

c. 向Class表中插入如下数据：

ClassNo Name TeacherNo
P611107 信息1班 T01
P611207 信息2班 T02
P611307 信息3班 T10

insert into class(classno,cname,teacherno) values('P611107','信息1班','T01'), ('P611207','信息2班','T02'),('P611307','信息3班','T10');

注意：由于在前面的T-SQL命令在Class表和Teacher表之间建立了外键引用关系，所以当插入第三条数据“T10”的时候出现错误。请说明为什么会出错。

AN：因为teacherno只定义到了3

d. 删除Class表、Teacher表中的数据

4．更新数据

（1）用Workbench或者Navicat将teacher表教师编号为’T01’的教师的姓名改为“赵六”。

请你自己写出SQL语句：

update student set name='赵六' where teacherno='T01';

（2）使用SQL语句，把教师编号为T02的教师的姓名改成“王五”。

请你自己写出SQL语句：

update student set name='王五' where teacherno='T02';

（3）把学号为“61130701”的学生的姓名改为“赵美”性别改成“女”。

请你自己写出SQL语句：

update student set name='赵美' and sex=’女’ where studentno='61130701';

（4）把所有学生的成绩加10分。

请你自己写出SQL语句：

update student set score=score+10;

（5）把学号为“61130701”，课程编号为C03的学生成绩改为80分。

请你自己写出SQL语句：

update student set score=80 where studentno=’61130701’ and courseno='C03';

5．删除数据

（1）把学号为“61140703”学生的成绩删除；

请你自己写出SQL语句：

delete from score where studentno='61140703';

请你自己写出SQL语句：
