实训二 ·二 数据库的建立和维护

一、实训目的与要求

通过实训，要求学生掌握在使用MySQL语句和图形化界面（WorkBench或Navicat）创建和管理数据库及表的方法。(1)创建和管理数据库。(2)创建表、确定表的主码和约束条件。

二、实训设备和软件环境

计算机、MySQL软件、WorkBench或Navicat软件

三、实训课时

2课时

四、实训内容**和结果记录

1、创建数据库school，默认字符集为utf8mb4，默认校验原则为utf8mb4_general_ci

2、根据以下表结构信息创建数据表

（1）教师表（表名英文：teacher）

列名 描述 数据类型 长度 是否允许空 默认值 说明
TeacherNo 教师编号 Char 3 N 无 主键
Tname 教师姓名 Varchar 10 N 无

（2）班级表（表名英文class）

字段名称 描述 数据类型 长度 是否允许空 默认值 说明
ClassNo 班级编号 Char 7 N 无 主键
Cname 班级名称 Varchar 20 N 无
TeacherNo 教师编号 Char 3 Y 无 外键Teacher

（3）学生表（表名英文student）

字段名称 描述 数据类型 长度 是否允许空 默认值 说明
StudentNo 学号 Char 8 N 无 主键
Sname 姓名 Varchar 10 N 无
Sex 性别 enum N 无 (‘男’,’女’)
Birthday 出生日期 Date N 无
ClassNo 班级编号 Char 7 Y 无 外键Class

（4）课程表（表名英文course）

列名 描述 数据类型 长度 是否允许空 默认值 说明
CourseNo 课程号 Char 3 N 无 主键
Coname 课程名 Varchar 20 N 无
Point 学分 int N 无

（5）成绩表（表名英文score）

列名 描述 数据类型 长度 是否允许空 默认值 说明
StudentNo 学号 Char 8 N 无 复合主键 外键Student
CourseNo 课程编号 Char 3 N 无 复合主键 外键Course
FinalScore 成绩 int Y 无
