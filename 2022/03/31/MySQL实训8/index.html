MySQL实训8-索引与视图

实训八 索引与视图

一、实训目的与要求

通过本次实训，使学生加深对索引和视图的理解，熟练掌握创建视图的命令。

二、实训设备和软件环境

计算机、MySQL软件、WorkBench或Navicat软件

三、实训课时

2课时

四、实训内容和结果记录

写成下列相关SQL语句

1、为了便于教师姓名查找，在teacher表的name列上建立一个升序的普通索引teacher_name；

create index teacher_name on tbl_teacher(tname);

2、在course表的name列上建立一个唯一索引course_name；

create unique index course_name on tbl_course(coursename);

3、在student表的studenno和classno列上建立一个复合索引stu_class；

create index stu_class on tbl_student(stuno,classno);

4、查看前面在三个表里创建的索引；

show index from tbl_teacher;

show index from tbl_course;

show index from tbl_student;

5、删除前面三个表里创建的索引；

drop index teacher_name on tbl_teacher;
drop index course_name on tbl_course;
drop index stu_class on tbl_student;

6、创建teacher1表，其表结构和teacher表一致，将其name字段建立一个唯一性索引t_name；

create table teacher1(tno char(3) not null primary key,tname varchar(10) not null,unique index t_name(tname));

7、创建一个基于A老师可以查看的视图A，取名为stu_view，要求查询输出w602319班学生的信息，并将字段改为中文；

create view stu_view as select classno as 班级号,classname as 班级名,tno as 班级代号 from tbl_class where classno='W602319';

8、创建一个视图stu_score_view，用于显示选修了C01课程的学生的学号、姓名、课程名、班级和成绩；

create view stu_score_view as select tbl_student.stuno,stuname,coursename,classno,score from tbl_student inner join tbl_score on tbl_score.stuno=tbl_student.stuno inner join tbl_course on tbl_course.courseno=tbl_score.courseno;

9、查看stu_score_view视图的基本信息；

describe stu_score_view;

10、查看视图stu_view的详细信息；

show table status like 'stu_view';

11、修改stu_view视图，只显示w602319班女生的信息；

alter view stu_view as select * from tbl_student where classno="P601151"and sex="女";

12、删除stu_view视图；

drop view stu_view;

13、新建一个检查视图stu_view，用来存放W602419班级学生信息，向该视图中添加一条记录（60231909,张天鹏,男, 2000-12-25,W602419），修改其班级为W602319，是否能修改，如果不能，为什么？如何才能修改其班级？删除该记录。

create view stu_view as select * from tbl_student where classno="W602451"; insert into stu_view values('60235126','张天鹏','男', '2002-12-25', 'W602451'); update stu_view set classno="w602351" where stuno="60235126";
