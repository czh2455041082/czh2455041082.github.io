日常-个人阶段目标设计

个人阶段目标设计

转眼已经是大一的第二个学期了，大二新学期要开始制定新的计划了。

在新计划制定前，先回顾一下上一次的目标阶段设计计划：上学期的课程顺利结束，这一学期 也积极参与了各项活动（志愿者，合唱比赛等），积极开展了阅读活动等。

然后来制定往后的阶段性计划：

首先是即将到来的暑假时间，暑假3个多月的假期，是一个很好的提升自我的机会。

1、 学习。学习是作为学生的第一要务。利用暑假时间复习大一所学基础知识。尤其是计算机实操方面的技术。加强专业课的训练，尤其是Python，Java。其次，开始准备专升本的学习任务，争取能够在大三考取一个更好的学校。

2、 生活。提升个人生活的能力与技巧，利用空闲时间积极参与社会和社会的志愿者活动，帮助他人，遵守法律。做一个遵纪守法的新时代青年。

3、 娱乐。专注于自身爱好的培养，注重阅读。疫情当下，要减少外出次数，在阅读中开拓视野。遨游知识的海洋，探索未知的神秘事物，让阅读充实精神思想。

然后是大二的学期目标：第二学期是大学学习生活的重要时期，所以目标制定会更加严谨详细。

1、 大二初期，学好专业课并且提高专业课技能素养。注重专升本课程高数和英语的学习，争取能有所收获

2、 大二期中，熟练掌握并且能运用自己所学的知识，能够利用自己所学做一些工作。争取能考过英语四级。

3、 大二期末，准备大三的实习工作，多去社会实践中锻炼自己，找到适合自己的工作。

4、 在学习工作中，积极参与志愿者活动，服务社会，结交朋友。

5、 学习生活更加紧张的同时，也要抽空开展阅读活动，放松自己。

日常-心理健康指南

心理健康指南

作者：珏

一 目录

Tips：

文章用时：2.5H

二 导言

2022年，转眼此事已经过了三年的时间，在这三年中，经历了许多事情。笔者原本高中满怀期待的高三毕业旅行也不得不取消。也因为这件事，原来畅通无阻的通行也变得艰难起来。各个高校也为了避免感染风险，制定了各种策略。

笔者这一届大一学生，还没有尝到大学“自由”的生活，就不得不“在校学习”。大家的心理或多或少也些许的“落差感”，因此，本篇心理健康指南就此诞生。

第一次尝试这种书本类型的书写格式，或许排版会有点小问题，见谅。

希望本篇指南对于处在迷茫，焦虑时期的读者有所帮助。

三 正文

1 导入：

1.1 概念

首先作为篇心理健康指南，我们首先要了解什么是心理健康。

2001年，世界卫生组织将心理将康定义为：“心理将康是一种健康或幸福状态，在这种状态下，个体可以实现自我，能够应对正常的生活压力，工作富有成效和成果，以及有能力对所在社会做出贡献。”[1]

1.2 标准

了解了心理健康的标准后，我们分八个标准来进行判断。

分别是：

1.智力正常 2.正确认识自我，接纳自我 3.情绪健康 4意志健全 5.人际关系和谐 6.人格完整统一 7.社会适应良好 8.心理行为符合年龄特征。[2]

针对此次是针对新冠疫情下大学生如何缓解心理压力，所以笔者通过专业媒体和自身角度对这个命题做出阐述。

2 央视新闻的建议[[3]](#_ftn3)：

我们先来看一下央视新闻给出的一些指导。具体看脚注3，不多展开。

2.1 总怀疑自己得新冠肺炎了，则么办？

先稳定情绪，仔细观察自身状况，和新冠肺炎的典型症状是否相符。身体不适，要及时就诊，在医生的指导下用药。

*对于在校学生，遇到健康码为非绿色，要及时联系班主任。

2.2 知道自己健康，但还是控制不住“疑病”。

尝试转移注意力，找一件事情让自己长时间投入去做（例如阅读[4]），必要时可以寻求心理治疗

*可以关注学校微信公众号：心灵驿站

3 笔者自身的经历、做法和建议

3.1 经历

笔者作为正在经历的一员，也曾有过心理的焦虑问题，或偶尔也会出现自心理问题，不过这是很正常的事情。所以结合自身经历来谈谈对这个问题的看法。用词可能没有那么专业，也可能会有点错误，见谅。

焦虑的存在。笔者认为焦虑是一直存在的，只不过疫情的存在，大家习惯了把焦虑都归咎于疫情这件事，而没有仔细的思考具体的原因。换句话说是惰性心理的作用，即现在常说的“摆烂心理”：找一个理由，不断地说服自己不去做别的事情。而疫情的出现正好成为了一个所谓的“正当理由”。

落差感。高中的高强度学习和对大学自由生活的憧憬，结果却因为疫情不能过上想象中的大学生活的落差。以及面对大学生活的一堆事情，幻想与现实的完全不符，造成的不满情绪。

以上俩点其实我认为都可以归咎在第二点里面，想象中大学生活应该是无拘无束，课程少，课程简单，自由时间多，可以做自己想做的事情。但现实是，有许多新的课程，课余时间也有很多琐碎的事情要做。能自我安排的时间有限。

其实想要解决也很简单，我们要正确的面对现在的生活，脱离“幻想生活”，做应该做的事情。

3.2 做法

3.2.1 关于课程：

笔者学的是计算机专业。高中时期都是理论课，进了大学，面对完全不同的课程内容和上课方式（上机实操），一下子难以适应。

因此笔者会尝试改变学习方式，在课余时间多做练习，多回顾与复习上课的内容，争取做到熟能生巧。

3.2.2 关于活动:

大学不同于高中，有各式各样的部门，社团和种类丰的活动。笔者刚进大学时期，面对课程的压力。我选择参与一些课余活动来丰富自己的业余生活。在操舞队锻炼身体，在辩论赛断论反应，参与志愿者活动服务他人，参与征文锻炼写作，开展讲座分享知识，参与义修提升技能……

院，校，社团开展的各种丰富有趣的活动是缓解压力的有力措施。

*多关注班级群中的通知和学校公众号的活动，总有适合自己的！

3.2.3 关于娱乐：

笔者作为一名游戏玩家，先从玩游戏角度来谈。游戏作为一种大众化，年轻化的娱乐方式。笔者认为适度的游戏确实是一种不错的缓解压力的娱乐措施，但一定要注意适度。不要“玩物丧志”，不要“再玩一把就睡觉”。

音乐作为一种老少皆宜的娱乐和缓解压力方式。但我们也要注意场合，外放注意音量，不在正式场合戴耳机听歌等不礼貌行为。

短视频：

如今短视频等流媒体平台的兴起，这种一直滑不到底的浏览方式越来越火爆，但一定要注意时长。给自己设定观看时长，避免滑着滑着几个小时就消失了的出现。

3.2.4 自己的活动：

无聊的时候做了做了自己的博客网站Zone | AlaxEleven和公众号。坚持每天写周记，即使是水文。

四 Tips

1.非必要不出远门，我们不知道何时何地会出现意外情况。

2.警惕电信诈骗，大学生依旧是电信诈骗的主要目标。一定要注意保管自己的财产。

五 结束语

第一次尝试多级列表的书写方式，多级分类可能有点不太好，见谅。这次的内容和排版没有达到预期的效果，略有遗憾。

说是指南，更多应该是经历的分享了，希望对你有所帮助。

面对疫情，我们首先要找准定位，学习依旧是第一要务，在学习的同时，积极参与活动，适当的娱乐项目是缓解压力的有力手段。如果有压力，请一定不要羞涩，寻找心灵驿站寻求帮助。

希望大家能好好学习，好好生活。

GitHub部署

一、创建一个GitHUb

如果已有GItHub账户，可以跳到第二步

打开GitHub官网：GitHub: Where the world builds software · GitHub

点击右上角Sign up

输入注册的邮箱

输入密码

输入用户名

是否接收咨询，我选择n 否

完成人机验证

到邮箱确认注册就完成了GitHub账号注册

二、创建个人仓库

在GitHub主页右上点击new，新建仓库界面

创建一个名为xxx.github.io的仓库（xxx为你的用户名），因为我这个已经创建，所以会提示已存在

点击下方绿色的Create repository 完成仓库的创建

三、添加SSH到GitHub

在刚才的博客文件夹，打开git界面，输入

git config --global user.name "yourname"
git config --global user.email "youremail"

yourname为你的GitHub名字，即上方的xxx

youremail为你注册GitHub的邮箱

检查是否正确，输入

git config user.name
git config user.email

创建ssh，输入

ssh-keygen -t rsa -C "youremail"

等待完成后到 本地磁盘C—用户-用户名 会看到一个.ssh的文件夹，打开里面的id_rsa.pub 可以选择用记事本打开

ctrl+a全选 ctrl+c复制

回到GitHub官网，点击右上角头像，点击Setting，找到右侧的SSH and GPS keys,点击右侧绿色的New SSH key,Title里面选输入这个Key的名字，我选择叫Hexo，在key里面 ctrl+v粘贴刚才复制的ssh。点击下方绿色的Add SSH key。

检查是否正确，输入

ssh -T git@github.com

Tips：第一次部署可能会出现验证，输入yes即可

四、部署Hexo到GitHub

回到hexo 文件夹 找到_config.yml 打开 我用的是vscode，没安装可以选择记事本

找到这几行代码

deploy:

type: git

repo: https://github.com/youname/yourname.github.io.git

branch: master

修改yourname为你的github名字，修改完保存退出。

在github中输入来安装deploy-git

npm install hexo-deployer-git --save

接着输入

hexo clean
hexo generate

ssh -T git@github.com（如果直接hexo d不行就加一句这个，出现successfully即可，需要挂T）

hexo deploy

hexo clean：清除了你之前生成的东西，也可以不加。
hexo generate ：生成静态文章，可以用 hexo g缩写
hexo deploy ：部署文章，可以用hexo d缩写

出现下图即为成功

补充：hexo d之后会弹出一个界面登录github，只有第一次会出现，然后按提示登录即可（要挂T）

五、使用

hexo d之后出现紫色git即为部署完成。

打开浏览器 博客的地址为 githubname.gtihub.io githubname为自己的github名字

hexo new “文章名字”：新建文章

.md书写规范教程写作 | Hexo

每次在本地写完文章输入hexo clean ,hexo g ,ssh -T git@github.com,hexo d 等待三分钟左右 即可打开博客看到更新的内容
